# Start

```bash
npm i
```

# Run in browser

```bash
npm start

// =>
// port: 3000
// open: http://localhost:3000
```

- custom port


```bash
npm start 5000

// =>
// port: 5000
// open: http://localhost:5000
```


# Run demo in shell
